## Pontuação
Cada projeto terá suas funcionalidades obrigatórias e desafios extras. O projeto obrigatório será o equivalente a uma pontuação final (que ainda será definida). A pontuação final receberá decrementos de acordo com a data de entrega.
- Se o projeto for entregue no dia (independente da hora), receberá a pontuação completa;
- Se o projeto for entregue dentro de até 3 dias, a pontuação será decrementada;
- Se o projeto for entregue após 3 dias e dentro de uma semana após a data definida do projeto, a pontuação será decrementada ainda mais;
- Se o projeto for entregue após a data final concederão nota 0;
- Para cada dia de atraso, a pontuação a ser entregue vai diminuindo até atingir 0;
- Cada desafio completo concederá uma pontuação extra, com sua soma total chegando ao valor máximo de 2,5pts e podendo ser dividida entre as disciplinas Projeto Integrador e Desenvolvimento de Aplicações Móveis.
